[h1]Mod Features [/h1]
[list]
[*] Admin Pistol - Can remove/destroy anything in-game, including ORP protected bases.
[*] Admin Pickaxes - A variety of in-game pickaxes that can be crafted that give boosted stats for farming.
[*] Admin Boosters - Can use boosters to enable admin related commands and effects in-game.
[*] Admin Radios - Can spawn in music radios that play in-game music.
[*] Prop Hunt Potions - These potions allow you to turn into a mesh in-game.
[*] Engram Unlocker - This allows you to unlock all the in-game engrams when consuming them.
[*] Tek Engram Unlocker - This allows you to unlock all the tek in-game engrams when consuming them.
[/list]
